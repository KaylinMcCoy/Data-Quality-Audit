
# Data Quality Audit

## Project Overview
This project was a data quality audit on the Retail data which includes the following tables (product, customer, sales, and store) where I found missing data. This led me tocleaning up the data issues, and then using Tableau visualization to showcase the results.

## Project Structure
- **sql_scripts/** - SQL exploration and data quality checks by table
- **Data_Quality_Governance_Report.md** - Full governance report
- **Tableau Dashboard** - Visualizations of findings

## Key Findings
- 3,829 data quality issues identified
- 100% remediation rate
- Consistent placeholder strategy implemented

## Technologies Used
- SQL (SQLite)
- Tableau Public
